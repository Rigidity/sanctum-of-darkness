# Sanctum of Darkness

A 2D top-down puzzle game made with Bevy.

# Installation

## If you have cargo installed:
clone the repo with 
```
git clone git@github.com:Rigidity/sanctum-of-darkness.git
```
and just run
```
cargo run
```

## If you do not have cargo installed:
install cargo using the instructions [here](https://www.rust-lang.org/tools/install)

I'd also recommend reading the [rust book](https://doc.rust-lang.org/book/) if you're unfamiliar with rust :)
and also the bevy docs, as the project uses bevy as the game engine: [https://bevy.org/learn/]
